Project folder: kernelsrc

Folder/File Name Description
boot Contains the GRUB multiboot assembly file.
grub Has the GRUB iso config number.
include Contains all the C headers for the kernel.
isodir Don't touch if you don't know what you're doing. Used for grub iso generation.
kernel Contains all the kernel source code.
obj Used to store object files during compilation.
Makefile Used to compile the kernel
linker.ld Linker script for the linker
setenv.sh Sets up the environment for you
build.sh Builds the kernel (make clean && make)
